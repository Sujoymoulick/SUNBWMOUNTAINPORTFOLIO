A high-performance, immersive portfolio experience built with **Next.js 15/16**, **React 19**, and **Three.js**. Featuring fluid animations, 3D voxel scenes, and interactive UI components.

## ✨ Key Features

- **Immersive 3D Visuals:** Utilizing `Three.js`, `OGL`, and `WebGL` for stunning backgrounds and voxel scenes.
- **Fluid Animations:** Smooth scrolling powered by `Lenis` and sophisticated animations using `GSAP` and `Framer Motion`.
- **Interactive UI:** Custom components like `CircularGallery`, `InfiniteMenu`, and `PixelCursorTrail`.
- **Responsive Design:** Fully optimized for all devices using **Tailwind CSS v4**.
- **Dynamic Content:** Showcasing projects, certifications, and a professional journey in a visually engaging way.

## 🛠️ Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Library:** [React 19](https://react.dev/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **3D Engine:** [Three.js](https://threejs.org/) & [OGL](https://github.com/o-gl/ogl)
- **Animation:** [GSAP](https://greensock.com/gsap/), [Framer Motion](https://www.framer.com/motion/)
- **Smooth Scroll:** [Lenis](https://github.com/darkroomengineering/lenis)
- **Icons:** [Lucide React](https://lucide.dev/)

## ⚙️ Getting Started

### Prerequisites

- Node.js >= 20.0.0
- npm / yarn / pnpm / bun

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Sujoymoulick/SUNBWMOUNTAINPORTFOLIO.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

### Build

To create an optimized production build:
```bash
npm run build
```
